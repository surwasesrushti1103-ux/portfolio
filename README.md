## Deploy

To deploy with GitHub Pages:

1. Create a GitHub repository under `https://github.com/surwasesrushti1103-ux`
2. Push this folder to the repository:
   ```powershell
   git remote add origin https://github.com/surwasesrushti1103-ux/<repo-name>.git
   git push -u origin main
   ```
3. On GitHub, go to `Settings` → `Pages`
4. Select `main` branch and `/ (root)` folder
5. Save and wait a few minutes

## Live URL

If the repo is named `portfolio`, the site will be available at:

`https://surwasesrushti1103-ux.github.io/portfolio/`
